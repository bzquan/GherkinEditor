Introduction

Gherkin is a simple language for writing specification by examples. Gherkin is plain-text English (or one of 60+ other languages) with a little extra structure. Gherkin is designed to be easy to learn by non-programmers, yet structured enough to allow concise description of examples to illustrate business rules in most real-world domains.



Keywords

In Gherkin, each line that isn't blank has to start with a Gherkin keyword, followed by any text you like. The main keywords are:

 

Feature

Scenario

Given, When, Then, And, But (Steps)

Background

Scenario Outline

Examples

 

There are a few extra keywords as well:

# (Comments)
