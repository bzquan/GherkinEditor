Simple Help

Features

  1. Syntax highlighting
  2. IntelliSense for Gherkin
  3. Dynamic pretty formatting Gherkin table
  4. Displaying image files in Gherkin document
  5. Mathematical formula in LaTex style
  6. Plotting Gherkin table data
  7. Internationalization supporting
  8. C++ code generation for executing Gherkin

Editing

Displaying images

You can use following markdown to display images files. The scale is a value between 10 to 500. It is optional and the default value is 100. And maxium width or height would be scaled down to 1024 Pixels. The tag "IMAGE" is case insensitive.
And the markdown can have an optional description.
#[IMAGE scale description](image path),e.g.
#[IMAGE 50 this is a Gherkin](Gherkin.png)
The base folder of the image is the folder where the Gherkin document is saved.It is possible to have a subfolder, e.g. Images/Gherkin.png.
Images will be showen in the editor located right of split view. This editor is called viewer editor.

Mathematical formula

You can use following markdown to describe TaTex mathematical formula. The scale is a value between 10 to 100. It is optional and the default value is 20. The tag "LaTex" is case insensitive.
And the markdown can have an optional description.
#[LaTex scale description](LaTeX typsetting), e.g.
#[LaTex 20 this is an equation](\sqrt{\frac{ax^2+bx+\pi}{x^3}}))
Mathematical formula will be shown in viewer editor.

Examples

  • Sum expression can be added using the lower and upper. For example, \sum_{lower}^{upper}

Plotting Gherkin table data

You can use following markdown to plot curve by using Gherkin table data. The next line must be starting of Gherkin table except an empty line and comment line. The tag "Plot" is case insensitive.
And the markdown can have an optional spline and description, which will be displayed as title of the plot.
#[Plot spline description](Table column name for X axis, Table column name for Y axis), e.g.
#[Plot spline this is sample graph](X Column, Y Column)
The plot size is 600 * 400 and will be shown in viewer editor.

Copying generated C++ code to a folder
  1. To check "Using unicode name in C++" if your C++ compiler support unicode symbols
  2. Generating C++ test code by using "Generate C++ test code" menu
  3. Copy generated C++ files to a folder by context menu in workspace window
How to convert to other format, e.g. PDF
  • You can use tool menu to save current document as PDF, XPS, RTF or Word file.
    To save as Word(docx) file you need Microsoft Word installed.
    To save as PDF by using Word you need Microsoft Word 2007 or above installed.
  • You can use "Microsoft Print To PDF" when printing to generate PDF file if you are using Windows 10. "Microsoft Print To PDF" would be installed by default in Windows 10.
  • You can use "Microsoft XPS Document Writer" when printing to generate XPS file
