Drawing sketch with Gherkin table

We can use Gherkin table to draw a sketch. A sketch is generated by using Graphviz

Therefore we need to install Graphviz and add Graphviz path to Gherkin editor's config file. For example,
<add key="graphVizLocation" value="C:\Program Files\Graphviz2.38\bin" />

Here is an example.

#[Graphviz]
Node and Edge Label Color
stub10->stub11 route 8 blue
stub11->stub20 route 9 green
stub11->stub12 route 8 green

By using #[Graphviz scale direction] comment line, we draw a sketch. For example,
#[Graphviz 80 TB]
 A sketch will be drawn from Top to Bottom in 80% scale by using Gherkin table.

Drawing directions

There are four drawing directions。LR(Left to Right, default), RL(Right to Left), TB(Top to Bottom)，BT(Bottom to Top)

complete format of Gherkin table for drawing a sketch

Node and Edge Shape Label Color Connections
n0 box text yellow n1, n2, n3

Node name will be displayed as label if node label is omitted. An edge without label will be displayed if label of edge is omitted. An label can be multi-line texts, e.g. 1st line\n 2nd line

Some supported node shapes

box, polygon, ellipse, point, oval, circle, doublecircle, triangle, tab, plaintext, diamond, trapezium, parallelogram, invtriangle, square, note, folder, rarrow, larrow, box3d

Edge directions

Example edge description
n1->n2 arrow from n1 to n2(n1→n2)
n1<->n2 bidirection arrow for n1 and n2(n1←→n2)
n1--n2 connect n1 and n2 without arrow(n1－n2)

Some edge shapes

diamond, crow, vee, dot, odot, onormal
